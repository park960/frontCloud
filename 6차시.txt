객체
데이터를 구조화하고 저장하기 위해 사용되는
키-값 쌍(key-value pair)으로 이루어진 데이터 구조이다

객체는 속성과 메서드를 가질 수 있으며,
속성은 데이터를, 메서드(함수 : 기능)는 동작을 나타낸다.
자바스크립트의 객체는 유연하고 동적으로 속성을 추가하거나 삭제할 수 있는 특징이 있다 .

##객체 데이터가 필요한 이유

현실 세계의 대부분의 데이터는 단순하지 않다
예를 들어, "맛동산"이라는 스낵을 생각해 보자
이 정보는 단지 이름만 있을까?
맛동산의 정보에는 여러 속성들이 포함될 수  있다

제조사: "오리온"
영양 정보: 칼로리, 지방, 단백질 등
무게: 100g
유통기한: 2026년 12월 31일


이처럼 다양한 속성을
하나의 단위로 묶어서 관리하려면
객체를 사용해야 한다
객체는 여러 개의 관련된 데이터를 구조화하는 도구로 활용된다.

C언어의 구조체를 통한 데이터 구조화
C 언어에서는 객체 개념이 없지만,
대신 구조체를 사용하여 데이터를 구조화할 수 있다
구조체는 여러 데이터를 하나의 단위로 묶어주는 역할을 하며, 객체와 비슷한 역할을 한다




##구조체 맛보기

#include <stdio.h>

// 구조체 정의
typedef struct {
char name[30];    // 스낵 이름
float price;      // 가격
int calories;     // 칼로리
int stock;        // 재고 수량
} Snack;

int main() {
// 스낵 정보 초기화
Snack snack = {"Choco Pie", 1.5, 250, 100};

// 스낵 정보 출력
printf("Snack Information:\n");
printf("Name: %s\n", snack.name);
printf("Price: $%.2f\n", snack.price);
printf("Calories: %d kcal\n", snack.calories);
printf("Stock: %d units\n", snack.stock);

return 0;
}
