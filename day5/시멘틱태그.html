시멘틱 태그 : 의미적 태그

div태그와 동일함 , 기능이 없음
div에 의미만을 부여함

시멘틱 태그는 html요소의 의미를 명확하게 나타내는 태그
시멘틱 태그를 사용하면 문서 구도를 이해하기 쉽고 검색엔진이 페이지의 컨텐츠를 더 잘 이해할수있고 또한 웹 접근성을 향상시키고 스타일링 및 레이아웃 작업을 보다 쉽게 수행할 수 있다.

  • header : 헤더영역
  • nav : 네비게이션 링크영역
  • main : 주요컨텍츠 (굳이)
  • article : 독립적인 콘텐츠 조각
  • sectin : 문서의 일반덕인 섹션, 그룹, 덩어리
  • aside : 주 내용과 연관성이 적은 컨텐츠 영역
  • footer : 문서의 바닥글, 저작줜정보, 연락처
